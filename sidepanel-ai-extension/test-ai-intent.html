üß† How the AI Intent System Works

  1. User Input: User types a message in the sidepanel
  2. AI Classification: System sends message to AI for intent analysis
  3. Context Analysis: AI considers current page, URL, and user context
  4. Intent Decision: AI returns classification with confidence and reasoning
  5. Smart Routing: System routes to appropriate handler based on AI decision
  6. Personalized Response: Handler uses AI's reasoning for better user communication
